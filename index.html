# input type description example
1 Text text input field for example we can use it for name
2 checkbox we can check more than one option for example we can use it to have extra hotel services internet tv radio
3 radio button we can only take one choice and we can use it for gender male female
4 password same of text but chars are encrypted
5 button used to run scripts after clicking
6 select used to rate a hotel for example
7 submit button used to send data to server
8 reset button used to reset the form
